# Aptos Nodejs的脚本代码分享

## 安装
1. `cp ./.env.example ./.env`
2. `npm i`
3. 配置.env中需要用到的内容


## scripts 脚本
### examples 示例脚本
1. bsc_transaction.js (官方示例)
2. simple_nft.js (官方示例)
3. transfer_coin.js (官方示例)
4. your_coin.js (官方示例)

### production 正式环境脚本
1. accounts.js (批量生成帐号并录入Mysql数据库)
2. liquidswap.js (Liquidswap官方Sdk的使用 - 一个生成交易信息并模拟提交，到签名提交的实用入门示例)
